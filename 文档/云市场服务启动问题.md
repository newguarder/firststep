## 云市场服务启动问题

1. 后端使用的是go语言。启动服务需要几点

   ```go
   docker ps -a
   
   docker start redis
   
   make test-run
   //完成启动后端go服务
   ```

2. cd到ui/management里面开启前端服务

   ```js
   npm run dev
   ```
